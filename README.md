# Supplejack Docker
Docker implementation of Supplejack stack (API, Manager and Worker). 

### Features
- Redis container
- Solr container
- Data container (Linked to host machine for development)
- Supplejack worker container
- Supplejack manager container
- Supplejack API container

### Prerequisites
- Dinghy
- Weave
- Docker Machine
- Docker Compose
- Virtualbox
